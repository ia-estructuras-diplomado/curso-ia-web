    Referencias y Lecturas¶

    📚 Libros¶

    Machine Learning¶

    • Goodfellow, Bengio, Courville (2016): "Deep Learning"
    • Biblia del Deep Learning

    • Disponible online: https://www.deeplearningbook.org

    • Hastie, Tibshirani, Friedman (2009): "The Elements of Statistical Learning"

    • Clásico en ML estadístico

    • Excelente para fundamentales

    • Aurélien Géron (2019): "Hands-On Machine Learning"

    • Práctico y actualizado
    • Código en Python

    Ingeniería y Estructuras¶

    • Russell & Norvig (2020): "AI: A Modern Approach"
    • Referencia en IA general
    • Capítulos sobre aplicaciones

    🎯 Roadmap de Aprendizaje¶

    Básico (1-2 meses)¶

    1. Python y Pandas
    2. Conceptos de ML
    3. Regresión y Clasificación
    4. Visualización

    Intermedio (2-3 meses)¶

    1. Redes Neurales
    2. Procesamiento de Datos
    3. Feature Engineering
    4. Validación de Modelos

    Avanzado (3+ meses)¶

    1. Deep Learning
    2. NLP / Computer Vision
    3. Series Temporales
    4. Aplicaciones específicas

    📝 Cómo Citar¶

    Para referencias en tus trabajos:

    Formato APA¶

    Autor(es) (Año). "Título del Paper". Publicación, volumen(issue), páginas. URL/DOI

    Formato Chicago¶

    Autor(es). "Título del Paper". Publicación, 2024.
