## 🎯 Objetivo

Aplicar conceptos fundamentales de Machine Learning para analizar datos reales de una estructura, identificando patrones y creando un modelo predictivo básico.

## 📊 Contexto

Tenéis datos de **sensores de un puente o edificio** con mediciones de:
- Desplazamientos
- Aceleraciones
- Temperaturas
- Estado clasificado manualmente

**Tu tarea:** Construir un modelo que **prediga el estado** de la estructura basándose en los datos de sensores.
