## 📝 Cómo Citar

Para referencias en tus trabajos:

### Formato APA
```
Autor(es) (Año). "Título del Paper". 
Publicación, volumen(issue), páginas.
URL/DOI
```

### Formato Chicago
```
Autor(es). "Título del Paper". 
Publicación, 2024.
```
